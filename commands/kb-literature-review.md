# /kb-literature-review

Use `obsidian-literature-workflow`.

Default outputs:
- `Knowledge/Literature Overview.md`
- `Knowledge/Method Taxonomy.md`
- `Knowledge/Research Gaps.md`
- `Writing/related-work-draft.md`
- `Maps/literature.canvas`
